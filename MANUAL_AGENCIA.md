# Manual do Usuário - Loquia (Agência)

## 1. Introdução

O Loquia é uma plataforma SaaS para gestão inteligente de campanhas publicitárias com IA integrada.

## 2. Primeiros Passos

### 2.1. Criar Conta
1. Acesse https://loquia.vercel.app
2. Clique em "Sign Up"
3. Preencha seus dados
4. Confirme seu email

### 2.2. Configurar Integrações
1. Acesse `/setup`
2. Clique em "Connect" em cada plataforma
3. Autorize o acesso
4. Aguarde o health-check

## 3. Dashboard

### 3.1. Visão Geral
- Lista de campanhas
- Métricas principais
- Alertas e notificações

### 3.2. Criar Campanha
1. Clique em "Nova Campanha"
2. Preencha os dados
3. Selecione as plataformas
4. Salve

## 4. Insights IA

### 4.1. Visualizar Insights
1. Acesse `/campaigns/[id]/insights`
2. Veja os insights gerados
3. Clique para ver detalhes

### 4.2. Reprocessar Insights
1. Clique em "Reprocessar Insights"
2. Aguarde a geração
3. Novos insights aparecerão

## 5. Otimizações IA

### 5.1. Revisar Otimizações
1. Acesse `/campaigns/[id]/optimizations`
2. Veja as otimizações sugeridas
3. Leia o impacto esperado

### 5.2. Aprovar/Rejeitar
1. Clique em "Accept" para aprovar
2. Clique em "Reject" para rejeitar
3. Após implementar, clique em "Mark as Complete"

## 6. Timeline

### 6.1. Visualizar Histórico
1. Acesse `/campaigns/[id]/timeline`
2. Veja todos os eventos
3. Filtre por tipo

## 7. Modo Cliente

### 7.1. Convidar Cliente
1. Acesse configurações
2. Clique em "Convidar Cliente"
3. Envie o link

### 7.2. Permissões
- Clientes podem visualizar tudo
- Clientes NÃO podem editar
- Clientes NÃO podem aprovar otimizações
